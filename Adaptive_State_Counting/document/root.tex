\title{Adaptive-State-Counting}
\author{Robert Sachtleben}
\maketitle


\begin{abstract}
    This entry provides a formalization of the adaptive state counting algorithm, used to test for reduction between finite state machines and originally presented by Hierons \cite{hierons} and slightly refined in \cite{refinement}.
    Definitions for finite state machines and adaptive test cases are given and many useful theorems are derived from these.
    The algorithm is first specified using mutually recursive functions, for which it is proven that the generated test suite is sufficient to test for reduction.
    Furthermore, the algorithm is also specified in a simple WHILE-language and its correctness is shown using Hoare-logic.
\end{abstract}
